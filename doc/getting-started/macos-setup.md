# macOS Setup

macOS is declared by the package and uses the Firebase Messaging support exposed by FlutterFire.

## Requirements

- Add the macOS target to your Flutter app
- Configure Firebase for macOS
- Enable the required entitlements and capabilities for push delivery

## Current Focus

macOS support is in active validation. Treat it as supported package surface, but verify token retrieval and real delivery on your target hardware before release.
