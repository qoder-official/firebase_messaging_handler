# Installation

Add the package to your app:

```yaml
dependencies:
  firebase_messaging_handler: ^1.0.0
```

Then fetch dependencies:

```bash
flutter pub get
```

## Minimum Environment

- Dart `>=3.5.0 <4.0.0`
- Flutter `>=3.3.0`
